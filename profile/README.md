# TrustC Project

Welcome to **TrustC Project**.

TrustC Project is an open-source organization dedicated to the design, development, and maintenance of the **TrustC programming language** and its **supporting development environments**, including compilers, verification tools, and related infrastructure.

Our goal is to advance trustworthy software systems by integrating modern programming language design, rigorous formal methods, and AI-assisted programming techniques.

## Support

The TrustC Project is supported by **Qiyuan Lab**, **Tsinghua University**, and collaborating academic and industrial partners.
